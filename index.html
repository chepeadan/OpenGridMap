About

We are an open community that crowdsources realistic power grid data to be used for research purposes. Here you will find the tools for crowdsourcing power grid data. The goal is to create an open platform for inferring realistic power grids based on actual data. Our vision is to provide a tool to researchers and practitioners that is able to produce realistic input data for simulation studies. OpenGridMap will support the entire process from data collection to formatting grid data for various purposes. We explore innovative ways to capture data and produce power grid approximations, e.g., using smartphone apps, drones, expert classification, existing map APIs, and graph inference algorithms. Join us and lets map the world's power grid!!! @opengridmap

How to help

  • Interested: Testing our App and giving feedback, Crowdsourcing data,

  • Researcher: Inference algorithms, Contribute to development (@opengridmap)

  • Utility: Distribution grid data sets for comparison with our inference, Rules and criteria used to design, distribution grids
